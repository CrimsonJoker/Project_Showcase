Basketball Timer

A basketball timer featuring configurable team names, game time and periods. Support overtime, with 24 and 14 seconds shot clock, a buzzer button and sound on game time and shot clock violations. With scoreboard. Added feature is an elapsed timer since the game started and a break timer when the game is paused, and this system is a Html, CSS, java script coded system.
